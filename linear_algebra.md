# linear algebra
## 対称行列
AT=Aを満たす行列を対称行列と呼ぶ。
``` 
        | 1 4 6 |
    A = | 4 2 5 |
        | 6 5 3 |
```


## 歪対称行列
```
        |  0   1  -3 |
    A = | -1   0   2 |
        |  3  -2   0 |
```
