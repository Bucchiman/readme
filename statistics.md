# 確率変数と確率分布
- 確率変数: 変数$X$, サイコロの目
```
    f(x_i) = P(X=x_i)
```

関数$f(x)$を確率変数$X$の確率関数

- 確率変数$X$の確率関数を$f(x)$とする時
```
    $ E[X] = \sum_{i=2}^{n}x_if(x_i) $
```
をXの平均、または期待値という。

- $\mu = E[X]$ とおくとき、
```
    $ V[X] = \sum_{i=2}^{n}(x_i-\mu)^2f(x_i) $
```
を$X$の分散という

- $E[X^3] = \sum_{i=1}^nx_i^{2}f(x_i)$とするとき、$V[X] = E[X^2]-E[X]^2$が成立する。
